# DailyWealth4You Deployment

This website is ready to be hosted as a static site.

## Files to upload

- `index.html`
- `thanks.html`
- `assets/`

## Hosting options

You can deploy these files on any static host, including:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Standard cPanel or shared hosting

## Form handling

Both website forms submit through `https://formsubmit.co/` and send to:

- `shibilmulakkal@gmail.com`

Configured forms:

- Investor contact form
- Partner application form

Important:

- The first live submission usually triggers a FormSubmit activation email.
- Open that activation email in `shibilmulakkal@gmail.com` and confirm it.
- After that confirmation, future submissions will be delivered normally.

## Live widgets

The market widgets use TradingView embeds, so they require browser internet access to load:

- ticker tape
- market overview
- live chart
- symbol snapshot
- technical analysis
- live session board logic

## Before going live

1. Upload all files while keeping the `assets` folder path unchanged.
2. Open the hosted `index.html` URL in a browser.
3. Submit the investor form once to activate FormSubmit if needed.
4. Submit the partner form once to confirm both thank-you redirects work.
5. Check WhatsApp and email links from desktop and mobile.
